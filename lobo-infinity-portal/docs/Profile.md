# Profile

Player profiles now include scheduling presentation data.

## League Identity

The permanent league key remains `Player`.

The portal may display a player display name, but all statistics, games, achievements, and scheduling lookups use the permanent `leaguePlayer`.

## Scheduling Fields

Profiles may display:

- Availability status.
- Preferred days.
- Preferred times.
- Discord handle.
- Schedule Match link.

These fields are presentation and coordination metadata. They do not affect standings, statistics, achievements, or historical game records.

## Editing

Players edit scheduling profile data from Match Finder.

My Profile includes a direct link to the availability editor.
