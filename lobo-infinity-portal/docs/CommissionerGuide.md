# Commissioner Guide

## Scheduling Oversight

Version 4.0 adds scheduling oversight to the Commissioner Dashboard.

Open:

`/commissioner`

Then expand:

`Scheduling`

The panel shows:

- Division completion percentage.
- Games remaining.
- Players behind pace.
- Outstanding matchups.
- Reminder candidates.

## Recommended Workflow

1. Review divisions behind pace.
2. Open reminder candidate profiles.
3. Encourage players to update availability.
4. Use Match Finder to identify likely pairings.
5. Avoid editing Google Sheets unless a production repair requires it.

## Permissions

Scheduling status is operations-gated.

Scheduling requests are participant-controlled unless an authorized operations user needs to assist.
