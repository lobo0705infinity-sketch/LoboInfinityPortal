# Lobo Infinity League Visual Brand Guidelines

## Brand Direction

The Lobo Infinity League interface should feel like a premium sci-fi league operations console: mercenary, industrial, tactical, tournament-ready, and readable on a phone during league night.

## Core Palette

- Tactical Black: `#030609`
- Command Panel: `#0D141C`
- Gunmetal Panel: `#121B25`
- Industrial Border: `#233244`
- HUD Blue: `#4CC9F0`
- Lobo Crimson: `#C1121F`
- Warning Amber: `#F0B13E`
- Promotion Green: `#2DC653`
- Relegation Red: `#EF233C`
- Primary Text: `#F5F8FB`
- Secondary Text: `#C4D0DA`
- Muted Text: `#90A0AF`

## Typography

Use a condensed, tactical heading style for brand, page, hero, and profile titles. Body text remains system sans-serif for speed and legibility.

## Shape Language

- Hexagonal crest and HUD motifs.
- 14-18px cards for readable touch targets.
- Thin cyan borders for operational panels.
- Crimson active states for key navigation.
- Amber only for warnings or record callouts.

## Motion

Motion should feel responsive, not decorative:

- 150-200ms hover lift on cards.
- Live status pulse.
- Skeleton loading for live-data waits.
- Respect `prefers-reduced-motion`.
